LDA Topic Modeling for User-Bot Chat

Using Python, I performed topic modeling on chatbot conversations with LDA. After vectorizing the text using TF-IDF, I extracted key terms and used Agglomerative Clustering to group related topics. The dendrogram helped reveal topic overlap, and I was able to surface common pain points across departments.

This topic extraction was used to recommend Knowledge Base (KB) article improvements and improve virtual agent design.
